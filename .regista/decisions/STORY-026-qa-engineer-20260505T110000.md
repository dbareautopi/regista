## Notas para el Developer

- La función `format_session_header()` es un placeholder que retorna `String::new()`. Todos los tests de contenido fallarán hasta que se implemente.
- La función debe internamente resolver modelos con `AgentsConfig::model_for_role()` para cada rol, construyendo el skill_path como `project_root.join(cfg.agents.skill_for_role(role))`.
- La función debe calcular `effective_max_iterations` como `max(story_count * 6, 10)` cuando `max_iterations == 0`.
- El timestamp ya se recibe como `&str` formateado — el caller se encarga de generarlo con `chrono::Utc::now()`.
