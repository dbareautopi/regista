### Solución requerida (QA)

Cada test necesita:
1. Cambiar `#[test]` → `#[tokio::test]`
2. Añadir `.await` tras la llamada a la función async

Ejemplo:
```rust
// Antes:
#[test]
fn process_story_accepts_shared_state() {
    let result = process_story(...);
    assert!(result.is_ok());
}

// Después (QA):
#[tokio::test]
async fn process_story_accepts_shared_state() {
    let result = process_story(...).await;
    assert!(result.is_ok());
}
```
