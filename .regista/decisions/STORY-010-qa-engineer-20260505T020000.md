## Decisión

Los tests para STORY-010 ya existen en `src/infra/agent.rs::tests` (14 tests en total) y cubren cada criterio de aceptación. Se verificó la cobertura uno a uno.

## Cobertura

| CA | Test(s) | Justificación |
|----|---------|---------------|
| CA1 | `invoke_once_is_async` | `.await` demuestra que `invoke_once` es `async fn` |
| CA2 | `invoke_once_respects_timeout` | Timeout 100ms, elapsed < 5s — demuestra que no hay busy-polling con `thread::sleep` |
| CA3 | `invoke_with_retry_is_async` | `.await` demuestra que `invoke_with_retry` es `async fn` |
| CA4 | `invoke_with_retry_preserves_retry_count`, `invoke_with_retry_backoff_doubles_delay` | Retry count + elapsed >= 1s con `retry_delay_base=1` — backoff exponencial con `tokio::time::sleep` |
| CA5 | `save_agent_decision_creates_file_with_content`, `save_agent_decision_noops_when_story_id_is_none`, `save_agent_decision_noops_when_decisions_dir_is_none` | 3 tests: happy path + 2 edge cases (no story_id, no decisions_dir). Todos llaman `.await`. |
| CA6 | `tokio_is_a_dependency` | `#[test]` sinónimo — compila solo si `tokio` está en `Cargo.toml` con features `time` y `process` |
| CA7 | `invoke_with_retry_fails_when_agent_not_installed` | Test ignorado adaptado de `#[test]` a `#[tokio::test] async` |
| CA8 | *(meta)* | `cargo test --lib agent` ejecutará todos los tests listados |

## Tests síncronos preservados (CA7)

- `build_feedback_prompt_includes_error` — función pura, sin I/O
- `build_feedback_prompt_truncates_long_output` — función pura, sin I/O
- `agent_options_default` — struct default

Estos tests no necesitan tokio y se mantienen como `#[test]`.

## Notas para el Developer

Los tests definen el contrato async. Para que compilen:

1. Añadir `tokio` a `[dependencies]` en `Cargo.toml` (o `[dev-dependencies]` si solo se usa en tests) con features: `rt`, `macros`, `process`, `time`, `fs`
2. Convertir `invoke_once()` → `async fn invoke_once()` usando `tokio::process::Command` y `tokio::time::timeout`
3. Convertir `invoke_with_retry()` → `async fn invoke_with_retry()` usando `tokio::time::sleep`
4. Convertir `save_agent_decision()` → `async fn save_agent_decision()` usando `tokio::fs::write` o `tokio::task::spawn_blocking`
5. Ejecutar `cargo test --lib agent` para verificar CA8
