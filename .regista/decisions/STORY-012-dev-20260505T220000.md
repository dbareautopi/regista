### Corrección necesaria (trabajo del QA)

Los 8 tests deben:
1. Cambiar `#[test]` a `#[tokio::test]`
2. Añadir `.await` en cada llamada a `process_story(...)` o `run_real(...)`

Ejemplo de la corrección esperada:

```rust
// Antes:
#[test]
fn process_story_accepts_shared_state() {
    let result = process_story(&story, tmp.path(), &cfg, &state, &agent_opts, &workflow);
    assert!(result.is_ok());
}

// Después:
#[tokio::test]
async fn process_story_accepts_shared_state() {
    let result = process_story(&story, tmp.path(), &cfg, &state, &agent_opts, &workflow).await;
    assert!(result.is_ok());
}
```

## Decisión

**NO se avanza a In Review.** Los tests no compilan. Corregir los tests es responsabilidad del QA. El orquestador debe transicionar Tests Ready → Tests Ready (transición #5: QA corrige tests) para que el QA actualice los tests con `#[tokio::test]` + `.await`.
